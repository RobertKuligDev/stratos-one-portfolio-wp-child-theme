# Stratos One Portfolio – Child Theme

A clean, modular, production‑ready WordPress child theme designed for a personal engineering portfolio.
Built with a focus on maintainability, modular CSS architecture, and block‑based patterns.

## 🚀 Features

- Modular CSS structure (`assets/css/`)
- Block patterns for each portfolio section
- Clean, minimal, production‑ready architecture
- No unnecessary dependencies
- Fully compatible with WordPress block editor
- Designed for extensibility and future premium add‑ons

## 📁 Directory Structure

```
stratos-one-portfolio/
├── assets/
│   └── css/
│       ├── hero.css
│       └── (more section styles soon)
├── patterns/
│   └── (block patterns)
├── functions.php
├── style.css
└── README.md
```

## 🧱 Development Principles

- **From general to specific** – global styles first, section styles later
- **Modular architecture** – each section has its own CSS file
- **Clean Git history** – meaningful, structured commits
- **No WordPress core files** – only the theme is versioned
- **Public‑safe** – no sensitive data, suitable for GitHub

## 🛠️ Installation

1. Place the theme in:
   `wp-content/themes/stratos-one-portfolio`
2. Activate it as a child theme in WordPress.
3. Patterns and styles load automatically.
